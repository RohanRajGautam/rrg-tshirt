# RRG T-Shirt 

This is an AI powered 3D website created using React, Node, Express, Tailwind, Three JS, and openAI API.


```
cd rrg-tshirt
docker build -t rrg-tshirt .
docker run -it -p 8000:80 rrg-tshirt
```

Go to: http://localhost:8000
